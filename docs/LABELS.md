# GitHub issue labels

| Label | Use |
|-------|-----|
| `bug` | Incorrect behavior or crash |
| `enhancement` | Feature request |
| `compat` | Broker or Go version interoperability |
| `protocol` | Kafka wire / API encoding |
| `integration` | Docker / CI integration tests |
| `serde` | Schema Registry, Avro, Protobuf, JSON Schema |
| `compat/kafka-4.x` | Kafka 4.x specific |
| `good-first-issue` | Small, well-scoped contributions |
| `security` | Security-sensitive (prefer private report per SECURITY.md) |
